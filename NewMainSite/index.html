Merging Android Security Updates

Every first monday of every month, Google push their source code for the latest update to Android. At the time of writing (Sat. 10th August 2019), the newest tag is: android-9.0.0_r46. For me to update my blueline to the latest source, I would need to go to https://android.googlesource.com/device/google/crosshatch and find the latest tag. As mentioned before, the latest tag at this time is android-9.0.0_r46 and my command space would consist of... cd device/google/crosshatch, and git pull https://android.googlesource.com/device/google/crosshatch android-9.0.0_r46. Once that is done, I will then git commit -m "--- Monthly Merge" and finalise everything with git push origin pasta.
